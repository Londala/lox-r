### Dev
Dev build:
`cargo build`

Run dev build:
`cargo run -- [options]`

### Release
Optimized release build:
`cargo build --release`

Run release build:
`cargo run --release -- [options]`

### Options
```
-i, --input-file <INPUT_FILE>  Path to the input file [default: <project-path>/src/lox-r/lox/input.lox]
-v, --verbose                  A flag to enable verbose mode
-h, --help                     Print help
```
